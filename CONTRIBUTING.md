# Contributing to INDEX Benchmark

누구나 PR 환영합니다.

## 기여 가능한 영역

- 감사 시나리오 추가 (새로운 사용자 유형, 서비스 유형)
- 감사 지표 제안 및 개선
- 챗봇/ARS 플랫폼 어댑터 추가
- 테스트 추가
- 번역 및 문서 개선

## 방법

1. Fork this repository
2. Create a branch: git checkout -b feature/my-contribution
3. Commit your changes
4. Open a Pull Request

## 주의

- INDEX 내부 구현 코드는 포함하지 마세요.
- 인터페이스/예시 중심으로 기여해주세요.
